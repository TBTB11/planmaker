# 入力フィールドリファレンス

フォーム実装時のフィールド仕様。各フィールドの型、例、備考を定義する。
凡例: **太字** = 必須、通常 = 任意（精度向上）

## 8. 入力頻度サマリー

| タイミング | 入力項目 | 所要時間目安 |
|-----------|---------|------------|
| 初回（生徒登録時） | 基本情報、目標、科目、スケジュール、単元リスト | 15〜30分 |
| 毎授業後 | 授業記録（単元、理解度、コメント）、進捗更新 | 3〜5分 |
| スケジュール変更時 | 振替、休講、曜日変更 | 1〜2分 |
| 季節講習前 | 講習設定（期間、回数、日程） | 5〜10分 |
| テスト後 | テスト結果、範囲、点数 | 2〜3分 |
| 月1回程度 | 目標・進捗の見直し、復習判定 | 5〜10分 |
