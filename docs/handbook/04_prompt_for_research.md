## コピー用プロンプト

```markdown
現在、以下の技術スタックで「ローカルファーストな学習計画管理アプリ」を開発しています。
この開発を効率化・自動化するために役立つ「ClaudeSkills（Claude向けのスキル定義）」、あるいは「MCP (Model Context Protocol) サーバー」、「Cursor向けのRules/Prompts」を探してください。

**プロジェクトの技術スタック:**
- **Frontend**: React, Vite, TypeScript
- **UI Framework**: Tailwind CSS, shadcn/ui
- **State Management**: React Hook Form, Zod
- **Database**: Dexie.js (IndexedDB wrapper)
- **Features**: Drag & Drop (dnd-kit), Local-first architecture

**知りたいこと:**
1. **ClaudeSkills / Prompts**: この構成（React + Dexie + shadcn/ui）での開発に特化した、コード生成やリファクタリング用のプロンプト定型文やスキルセットはありますか？
2. **MCP Servers**: ローカルDB (IndexedDB/Dexie) のデバッグや、Reactコンポーネントの管理に役立つMCPサーバーは公開されていますか？
3. **Best Practices**: この技術スタックでAIペアプログラミングをする際に有効な、公開されている`.cursorrules`やシステムプロンプトの例があれば教えてください。

URLや具体的な設定例も含めて教えていただけると助かります。
```
